# C++

Algorithms, language experiments, and small C++ projects.

## Layout

- **`algo/`** — Competitive programming (LeetCode, Codeforces, Advent of Code) and reusable snippets in `common/`.
- **`lang/`** — Standalone demos: caches, memory pool, STL/coroutines, C++20 samples.
- **`projects/`** — Slightly larger exercises (for example trading or exchange toy code).

Typical contest solutions use `#include <bits/stdc++.h>` and compile with a recent `g++` or `clang++`. CMake subprojects set their own standard (often C++17–23).

See the repository root [README](../README.md) for the rest of this monorepo.
