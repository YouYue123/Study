# Language experiments

Self-contained C++ demos and benchmarks, usually with CMake.

- **`cache/`** — Cache eviction policies and related tests.
- **`memory_pool/`** — Thread-aware pool allocator (`include/`, `src/`, `tests/`).
- **`STL/`** — Implementations or exercises mirroring standard facilities.
- **`C++20/`** — Language feature demos (e.g. `coroutine/`).
- **`virtual_inheritance/`** — Small inheritance model experiment.
